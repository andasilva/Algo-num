Mode d'emploi

Afin de pouvoir utiliser cet outil, il est indispensable de déposer le fichier de matrice dans le dossier "json" situé dans le même répertoire que cette page.

De plus, le fichier de matrice doit être au format 'json' et organisé de la manière suivante :

				{
					"n": [3],
					"A": [3.0,4.0,-1.0,1.0,-1.0,2.0,2.0,3.0,4.0],
					"B": [23.0,3.0,7.0]
				}
